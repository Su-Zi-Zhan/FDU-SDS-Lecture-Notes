## 前言

- 为什么做这个仓库：学期初那会心情不是很好，想写点东西麻痹一下自己；后来想断更的时候发现大家似乎确实需要这份笔记，于是便继续更新下去了。

  - 注重几何直观，有丰富的可视化。
  - $L_AT^EX$ 排版，格式尽可能规范。
- 使用的 $L_AT^EX$ 编辑器：VS Code（一年前我会嫌弃 VS Code 难用，但是大人，时代变了！AI 自动补写是真香！）
